После создания основного каркаса (задания нумерации stringer frame), следует собрать и занести информацию по узловым точкам. Помимо координат необходимо указать номер шпангоута, номер стрингер и сторону.

  1. Откройте меню "Внести данные"
  2. Выберите таблицу "Node"
  3. Задайте уникальные номер узла следую правилу Правило
  4. Укажите какому шпангоуту, стрингеру он принадлежит и с какой стороны находится
  5. Укажите номер используемой системы координат. Оставьте поле пустым для глобальной СК.
  6. Укажите координаты в выбранной системе координат
  7. Нажмите кнопку отправить, чтобы сохранить узел в БД
  8. Для пакетной загрузки узлов в БД, рекомендуется делать это через эксель. Нажмите чекбокс, скачайте шаблон для заполнения и следуйте инструкциям на листе Readme.
